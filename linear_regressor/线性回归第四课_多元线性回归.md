# 线性回归第四课：多元线性回归

这一节开始，我们把一元线性回归正式扩展到多个特征。

前面的一元线性回归是：

```text
y_hat = w x + b
```

现在如果一个样本不只一个特征，而是有多个特征，例如：

```text
x_1 = 学习时间
x_2 = 睡眠时间
x_3 = 练习题数量
```

那么模型就要变成：

```text
y_hat = w_1 x_1 + w_2 x_2 + w_3 x_3 + b
```

这就是多元线性回归。

## 1. 多元线性回归在做什么

一元线性回归里，模型只看一个特征。

多元线性回归里，模型会同时看多个特征，并学习每个特征对应的权重：

```text
w_1 -> 第 1 个特征的重要程度
w_2 -> 第 2 个特征的重要程度
w_3 -> 第 3 个特征的重要程度
...
```

所以多元线性回归不是“换了个公式”，而是把：

```text
一个特征的线性关系
```

扩展成：

```text
多个特征共同作用的线性关系
```

## 2. 单个样本的写法

假设某个样本有 `n` 个特征：

```text
x = [x_1, x_2, ..., x_n]
```

对应权重是：

```text
w = [w_1, w_2, ..., w_n]
```

那么预测值就是：

```text
y_hat = w_1 x_1 + w_2 x_2 + ... + w_n x_n + b
```

这其实就是“特征向量和权重向量的点积，再加偏置”：

```text
y_hat = x · w + b
```

## 3. 为什么会写成 Xw

前面那条式子是“单个样本”的写法。

如果把所有样本一起写，就要把数据组织成矩阵。

假设有 `m` 个样本、`n` 个特征：

```text
X.shape = (m, n)
```

也就是：

```text
X =
[
  [x_11, x_12, ..., x_1n],
  [x_21, x_22, ..., x_2n],
  ...
  [x_m1, x_m2, ..., x_mn]
]
```

参数向量：

```text
w =
[
  w_1,
  w_2,
  ...
  w_n
]
```

那么所有样本的预测值可以统一写成：

```text
y_hat = Xw + b
```

如果把偏置也并入矩阵，就会写成：

```text
y_hat = X_design theta
```

其中：

```text
X_design = 在 X 左边补一列全 1
theta    = [b, w_1, w_2, ..., w_n]^T
```

这就是黑马课程里“为什么转换时是 Xw”的核心原因：

```text
X 的每一行是一个样本
w 的每个分量对应一个特征权重
矩阵乘法会自动把每一行样本和权重做点积
```

## 4. 多元线性回归和一元线性回归的关系

可以把一元线性回归看成多元线性回归的特例。

一元时：

```text
y_hat = w x + b
```

多元时：

```text
y_hat = w_1 x_1 + w_2 x_2 + ... + w_n x_n + b
```

如果 `n = 1`，多元公式自然退化回一元公式。

所以：

```text
一元线性回归 = 只有一个特征的多元线性回归
```

## 4.1 一元和多元到底差在哪里

这个点非常重要，也最容易在刚学矩阵形式时混淆。

### 从单个样本看

一元线性回归：

```text
x 是一个数
y_hat = w x + b
```

多元线性回归：

```text
x 是一个特征向量
y_hat = x · w + b
```

所以从单个样本角度说：

```text
一元和多元的真正区别，是“每个样本有几个特征”
```

### 从整个数据集看

整个数据集写成矩阵时：

一元线性回归：

```text
X.shape = (m, 1)
```

多元线性回归：

```text
X.shape = (m, n)
```

所以从整体角度说：

```text
X 在一元和多元里都可以写成矩阵
区别只是列数不同
```

也就是说，更准确的表达不是：

```text
“多元时 x 才变成矩阵”
```

而是：

```text
“只要从所有样本整体出发，X 就可以写成矩阵；
一元是 1 列，多元是 n 列”
```

### 参数向量是否包含偏置，要看模型写法

如果模型写成：

```text
y_hat = Xw + b
```

那么：

```text
w -> 只包含各特征的权重
b -> 单独的偏置
```

如果模型写成：

```text
y_hat = X_design theta
```

那么：

```text
theta -> 包含偏置和所有权重
```

即：

```text
theta = [b, w_1, w_2, ..., w_n]^T
```

所以你以后看到公式时，要先问自己：

```text
当前写法里，偏置 b 是单独放着，还是已经并入参数向量了？
```

## 5. 多元线性回归的损失函数

和一元完全一样，只是预测值从 `wx+b` 变成了 `Xw+b` 或 `X_design theta`。

平方误差和：

```text
SSE = Σ_{i=1}^m (y_i - y_hat_i)^2
```

矩阵形式：

```text
SSE = ||y - X_design theta||_2^2
```

也就是说，多元线性回归仍然是在做最小二乘：

```text
min_theta ||y - X_design theta||_2^2
```

## 6. 多元线性回归的正规方程

一元线性回归推导完以后，我们得到了：

```text
X_design^T X_design theta = X_design^T y
```

这个式子对多元线性回归同样成立。

所以多元线性回归的正规方程仍然是：

```text
X^T X w_hat = X^T y
```

或者把偏置并进去后写成：

```text
X_design^T X_design theta_hat = X_design^T y
```

如果可逆：

```text
theta_hat = (X_design^T X_design)^(-1) X_design^T y
```

你会发现，一元和多元在线性代数结构上并没有变，只是：

```text
标量 -> 向量
二维小矩阵 -> 更大的矩阵
```

## 7. 几何意义没有变

多元线性回归的几何意义和一元是同一套逻辑：

```text
X 的列向量张成一个列空间
真实标签 y 通常不在这个列空间里
线性回归寻找列空间里离 y 最近的那个向量
```

也就是说：

```text
y_hat = Xw
```

是 `y` 在 `Col(X)` 上的正交投影。

最优残差：

```text
r = y - y_hat
```

满足：

```text
X^T r = 0
```

## 8. 多元线性回归的权重怎么理解

在多元线性回归里，每个权重都表示“其他特征保持不变时，该特征变化对预测值的线性影响”。

例如：

```text
y_hat = 20 + 8*x_1 + 3*x_2 + 0.5*x_3
```

可以理解成：

```text
x_1 每增加 1，预测值平均增加 8
x_2 每增加 1，预测值平均增加 3
x_3 每增加 1，预测值平均增加 0.5
```

但要注意：

```text
不同特征单位不同，系数大小不能总是直接横向比较
```

这也是为什么标准化在某些线性模型里很有帮助。

## 9. 多元线性回归和 NumPy shape

这一节特别要注意形状：

```text
X.shape        = (样本数 m, 特征数 n)
w.shape        = (特征数 n,)
y_hat.shape    = (样本数 m,)
```

如果并入偏置：

```text
X_design.shape = (m, n + 1)
theta.shape    = (n + 1,)
```

所以：

```python
y_pred = X @ w
```

或者：

```python
y_pred = X_design @ theta
```

这是后面读代码和手推公式时最重要的结构感。

## 10. 和 sklearn 的对应关系

在 `sklearn` 中，多元线性回归和一元线性回归用的是同一个模型类：

```python
from sklearn.linear_model import LinearRegression
```

区别只在于 `X` 的列数不同：

```text
一元：X.shape = (m, 1)
多元：X.shape = (m, n)
```

训练后：

```python
model.coef_
model.intercept_
```

其中：

```text
coef_      -> 每个特征对应的权重
intercept_ -> 偏置 b
```

## 11. 本节总结

```text
1. 多元线性回归是在多个特征上建立线性关系。
2. 单个样本的预测是“特征向量和权重向量的点积 + 偏置”。
3. 所有样本的预测统一写成 y_hat = Xw + b。
4. 把偏置并入矩阵后，可以统一写成 y_hat = X_design theta。
5. 多元线性回归仍然是在最小化 ||y - X theta||_2^2。
6. 正规方程形式没有变，只是矩阵维度变大了。
7. 几何意义仍然是把 y 投影到 X 的列空间。
```

## 10. 正规方程法存在的问题

1. 运算量过大：正规方程需要计算矩阵的逆，时间复杂度为 O(n^3)，当特征数 n 很大时，计算量会非常大。栈溢出和内存不足的风险也会增加。

2. 数值稳定性问题：当特征矩阵 X 的列之间存在高度相关性时，X^T X 可能接近奇异矩阵，导致求逆不稳定。
