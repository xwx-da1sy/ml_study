# 线性回归第二课：损失函数

## 1. 这一节要解决什么问题

第一课里我们已经知道：

```text
线性回归要学习一组参数
让预测值 y_hat 尽量接近真实值 y
```

这一节要继续追问：

```text
什么叫“尽量接近”？
怎么判断一条直线比另一条直线更好？
```

答案就是：

```text
看误差，也就是损失函数。
```

## 2. 预测值和残差

一元线性回归公式：

```text
y_hat = wx + b
```

对每个样本来说：

```text
真实值      y
预测值      y_hat
残差        e = y - y_hat
```

例如：

```text
真实分数 y = 80
预测分数 y_hat = 76
```

那么残差是：

```text
e = 80 - 76 = 4
```

如果残差为正，说明模型预测偏低；如果残差为负，说明模型预测偏高。

## 3. 为什么不能直接把残差相加

如果只把残差直接相加，会发生正负抵消：

```text
样本 1 的残差： 10
样本 2 的残差：-10

总和 = 0
```

但这并不意味着模型没有误差。

所以线性回归通常不直接用：

```text
sum(y - y_hat)
```

而是使用绝对值或平方。

## 4. 常见误差量

### 4.1 绝对误差

单个样本的绝对误差：

```text
|y - y_hat|
```

它只看误差大小，不看正负。

### 4.2 平方误差

单个样本的平方误差：

```text
(y - y_hat)^2
```

平方后也不会相互抵消，而且大误差会被放大。

## 5. 四个最常见的回归指标

### 5.1 SSE

平方误差和：

```text
SSE = sum((y - y_hat)^2)
```

含义：

```text
把所有样本的平方误差加起来
```

### 5.2 MSE

均方误差：

```text
MSE = mean((y - y_hat)^2)
```

也就是：

```text
MSE = SSE / n
```

其中 `n` 是样本数。

### 5.3 RMSE

均方根误差：

```text
RMSE = sqrt(MSE)
```

它的好处是：

```text
RMSE 的单位和原始标签 y 一样
```

例如房价预测中，如果 `y` 的单位是“万元”，那么 RMSE 也是“万元”。

### 5.4 MAE

平均绝对误差：

```text
MAE = mean(|y - y_hat|)
```

它不像平方误差那样特别强调大误差，因此对异常值相对更稳一些。

## 6. 为什么线性回归最常用平方误差

最常见原因有两个：

```text
1. 平方后不会正负抵消
2. 数学上更容易求导、推导最优解
```

第二点尤其重要，因为后面无论是正规方程还是梯度下降，都要利用这个性质。

## 7. 线性代数写法

设：

```text
y         -> 真实标签向量
y_hat     -> 预测向量
r = y - y_hat -> 残差向量
```

如果线性回归写成：

```text
y_hat = Xw
```

那么残差向量就是：

```text
r = y - Xw
```

平方误差和可以写成：

```text
SSE = ||y - Xw||_2^2
```

均方误差可以写成：

```text
MSE = (1/n) * ||y - Xw||_2^2
```

这里的 `|| · ||_2` 是二范数。

这一点很重要，因为它把“很多个样本的误差”统一成了“一个向量的长度”。

## 8. 损失函数的几何意义

从线性代数角度看：

```text
Xw          -> 列空间中的一个向量
y           -> 真实标签向量
y - Xw      -> 从预测点指向真实点的残差向量
```

线性回归做的事就是：

```text
在 X 的列空间里找一个向量 Xw
让它和 y 的距离最小
```

这个“距离最小”在平方误差下就是：

```text
min ||y - Xw||_2^2
```

所以：

```text
损失函数不是额外的新概念
它就是“预测向量离真实向量还有多远”的量化方法
```

## 9. 一个直观例子

假设真实值和预测值分别是：

```text
y      = [35, 45, 54, 65, 74]
y_hat  = [35, 44.8, 54.6, 64.4, 74.2]
```

残差向量：

```text
r = y - y_hat
  = [0, 0.2, -0.6, 0.6, -0.2]
```

那么：

```text
绝对误差 = [0, 0.2, 0.6, 0.6, 0.2]
平方误差 = [0, 0.04, 0.36, 0.36, 0.04]
```

于是：

```text
SSE  = 0.80
MSE  = 0.16
RMSE = 0.40
MAE  = 0.32
```

这说明模型已经拟合得比较好了，因为平均误差很小。

## 11. 本节总结

```text
1. 残差是 y - y_hat。
2. 误差不能直接相加，因为会正负抵消。
3. 回归中常见指标有 SSE、MSE、RMSE、MAE。
4. 线性回归最常用平方误差，因为它便于推导和优化。
5. 线性代数里，损失函数就是残差向量长度的平方。
6. 线性回归的目标是最小化 ||y - Xw||_2^2。
```
