# 线性回归第五课：梯度下降

这一节开始，我们学习线性回归的另一条解法：梯度下降。

前面已经学过正规方程。正规方程的特点是：

```text
直接解出最优参数
```

而梯度下降的特点是：

```text
不是一步算出答案
而是从一个初始参数出发，一点点逼近最优解
```

这也是黑马课程在线性回归章节里的下一步：

```text
一元线性回归：正规方程法
-> 多元线性回归：正规方程法
-> 单变量梯度下降法
-> 多变量梯度下降法
```

## 1. 为什么还要学梯度下降

你可能会问：

```text
既然正规方程已经能直接解出参数，
为什么还要多学一种方法？
```

原因主要有两个：

```text
1. 特征数很多时，正规方程求逆代价很高
2. 后面的逻辑回归、神经网络、深度学习大量依赖梯度下降
```

所以梯度下降不只是线性回归的一种解法，它是后面很多模型的通用优化方法。

## 2. 梯度下降的直觉：下山

把损失函数想象成一座山：

```text
山越高 -> 损失越大
山越低 -> 损失越小
山谷最低点 -> 最优参数
```

梯度下降做的事情就是：

```text
站在当前参数位置
看哪边下降最快
往那个方向走一小步
重复很多次
```

所以：

```text
梯度       -> 告诉你“上升最快”的方向
负梯度     -> 告诉你“下降最快”的方向
```

## 3. 学习率是什么

每次往下降方向走的时候，还要决定“走多大一步”。

这个步长就叫学习率，常记作：

```text
alpha
```

更新公式会写成：

```text
新参数 = 旧参数 - alpha * 梯度
```

学习率太小：

```text
下降很慢
训练很多轮还到不了低点
```

学习率太大：

```text
可能一下跨过最低点
来回震荡
甚至发散
```

所以学习率是梯度下降里最关键的超参数之一。

## 4. 一元线性回归的目标函数

一元线性回归：

```text
y_hat_i = w x_i + b
```

损失函数我们前面已经写过：

```text
J(w, b) = (1 / 2m) Σ_{i=1}^m (y_i - w x_i - b)^2
```

这里写成 `1 / 2m` 的形式，是因为后面求导更整洁。

## 5. 对损失函数求偏导

### 5.1 对 w 求偏导

```text
dJ/dw = -(1 / m) Σ_{i=1}^m x_i (y_i - w x_i - b)
```

### 5.2 对 b 求偏导

```text
dJ/db = -(1 / m) Σ_{i=1}^m (y_i - w x_i - b)
```

这两个偏导数告诉我们：在当前参数位置下，损失函数对 `w` 和 `b` 的变化有多敏感。

## 6. 梯度下降更新公式

既然目标是往“下降最快”的方向走，那么就要朝负梯度方向更新：

```text
w := w - alpha * dJ/dw
b := b - alpha * dJ/db
```

把偏导数代进去，就得到一元线性回归的更新公式：

```text
w := w + alpha * (1 / m) Σ_{i=1}^m x_i (y_i - w x_i - b)
b := b + alpha * (1 / m) Σ_{i=1}^m (y_i - w x_i - b)
```

这里看起来是“加号”，不是“减号”，是因为前面的偏导数本身带了负号。

## 7. 梯度下降的训练流程

一元线性回归的梯度下降训练流程可以记成：

```text
1. 随机初始化 w 和 b
2. 用当前 w、b 计算所有样本的预测值
3. 计算损失函数 J(w, b)
4. 计算 dJ/dw 和 dJ/db
5. 按更新公式更新 w 和 b
6. 重复很多轮，直到损失基本不再下降
```

## 8. 多元线性回归中的梯度下降

如果是多元线性回归，写成矩阵形式：

```text
y_hat = Xw + b
```

或者把偏置并入设计矩阵：

```text
y_hat = X_design theta
```

损失函数：

```text
J(theta) = (1 / 2m) ||y - X_design theta||_2^2
```

梯度可以统一写成：

```text
grad = (1 / m) X_design^T (X_design theta - y)
```

更新公式：

```text
theta := theta - alpha * grad
```

这就是多变量梯度下降的统一形式。

## 9. 梯度下降和正规方程的区别

正规方程：

```text
直接解
theta_hat = (X^T X)^(-1) X^T y
```

梯度下降：

```text
迭代逼近
theta := theta - alpha * grad
```

两者的区别可以先这样记：

```text
正规方程：一次性算答案
梯度下降：一轮一轮逼近答案
```

## 10. 梯度下降训练时你最该观察什么

学习阶段最重要的不是一下子背下公式，而是观察三件事：

```text
1. 损失是不是在持续下降
2. 参数 w、b 是不是逐渐稳定
3. 学习率变化会不会影响收敛速度和稳定性
```

如果损失不断下降并趋于稳定，说明梯度下降在正常工作。

## 11. 本节总结

```text
1. 梯度下降是一种迭代优化方法。
2. 它的直觉是“沿着负梯度方向下山”。
3. 学习率 alpha 决定每次走多大一步。
4. 一元线性回归里要分别更新 w 和 b。
5. 多元线性回归里可以统一写成 theta := theta - alpha * grad。
6. 梯度下降和正规方程是两条不同的求解路线。
```
