# 线性回归第一课

## 1. 当前学习位置

当前路线：

```text
机器学习概述
-> KNN 分类与回归
-> 标准化
-> Iris 案例
-> 交叉验证与网格搜索
-> 线性回归
```

从这一节开始，模型不再像 KNN 那样“预测时找邻居”，而是要从训练数据中学习一组参数。

KNN 的思路：

```text
新样本 -> 找最近的 K 个邻居 -> 投票或求平均
```

线性回归的思路：

```text
训练数据 -> 学习 w 和 b -> 用公式预测新样本
```

## 2. 线性回归解决什么问题

线性回归用于预测连续数值。

例如：

```text
学习时间 -> 考试分数
房屋面积 -> 房价
广告投入 -> 销售额
温度、湿度、风速 -> 用电量
```

这些目标值不是类别，而是可以连续变化的数值，所以属于回归问题。

## 3. 一元线性回归

如果只有一个特征，例如用学习时间 `x` 预测考试分数 `y`，模型可以写成：

```text
y_hat = wx + b
```

其中：

```text
x      -> 输入特征
y      -> 真实标签
y_hat  -> 模型预测值
w      -> 权重，也可以理解为斜率
b      -> 偏置，也可以理解为截距
```

如果：

```text
y_hat = 8x + 27
```

含义是：

```text
学习时间每增加 1 小时，预测分数平均增加 8 分。
当学习时间为 0 时，模型预测分数为 27 分。
```

## 4. 多元线性回归

如果有多个特征，例如：

```text
x1 = 学习时间
x2 = 睡眠时间
x3 = 练习题数量
```

模型写成：

```text
y_hat = w1*x1 + w2*x2 + w3*x3 + b
```

每个特征都有一个对应的权重。

## 5. 线性代数写法：y_hat = Xw

为了把所有样本一起计算，机器学习会把特征组织成矩阵。

假设有 4 个样本，每个样本 1 个特征：

```text
X =
[
  [1.0],
  [2.0],
  [3.0],
  [4.0]
]
```

这就是：

```text
X.shape = (4, 1)
```

也就是：

```text
4 个样本
每个样本 1 个特征
```

如果暂时不考虑偏置 `b`，预测可以写成：

```text
y_hat = Xw
```

如果要把偏置 `b` 也写进矩阵，可以给 `X` 增加一列全 1：

```text
X_design =
[
  [1, 1.0],
  [1, 2.0],
  [1, 3.0],
  [1, 4.0]
]

theta =
[
  b,
  w
]
```

于是：

```text
y_hat = X_design theta
```

这就是线性回归和线性代数最直接的连接。

## 5.1 单个样本和整个数据集要分开看

这里特别容易混淆，所以单独记一下。

### 单个样本的角度

一元线性回归时：

```text
x -> 一个数
y_hat = w x + b
```

多元线性回归时：

```text
x -> 一个特征向量
y_hat = x · w + b
```

也就是说：

```text
一元：单个样本只有 1 个特征
多元：单个样本有多个特征
```

### 整个数据集的角度

不管是一元还是多元，只要把所有样本一起写，`X` 都是矩阵。

一元线性回归：

```text
X.shape = (m, 1)
```

多元线性回归：

```text
X.shape = (m, n)
```

所以更准确地说：

```text
不是“多元时 x 才变成矩阵”
而是“从整个数据集角度看，X 一直都可以写成矩阵”
```

区别只是：

```text
一元：只有 1 列特征
多元：有多列特征
```

### 参数里是否包含偏置，也要分开看

如果写成：

```text
y_hat = Xw + b
```

那么：

```text
w -> 只包含特征权重
b -> 单独的偏置
```

如果写成：

```text
y_hat = X_design theta
```

那么：

```text
theta -> 包含偏置和所有特征权重
```

也就是：

```text
theta = [b, w_1, w_2, ..., w_n]^T
```

这一点非常重要，因为以后看到 `w` 时，要先判断：

```text
它是“只有权重”的参数向量
还是“连偏置一起合并进去”的参数向量
```

## 6. 残差

预测值通常不会和真实值完全一样。

真实值与预测值的差叫残差：

```text
e = y - y_hat
```

如果：

```text
y = 80
y_hat = 76
```

那么：

```text
e = 80 - 76 = 4
```

残差为正，说明模型预测低了；残差为负，说明模型预测高了。

## 7. 为什么要用平方误差

如果直接把残差相加，正负误差会互相抵消。

例如：

```text
10 + (-10) = 0
```

但这并不说明模型没有误差。

所以线性回归常用平方误差：

```text
(y - y_hat)^2
```

所有样本的平方误差求平均，就是均方误差 MSE：

```text
MSE = mean((y - y_hat)^2)
```

线性代数写法：

```text
MSE = (1/m) * ||y - Xw||_2^2
```

这里的 `||y - Xw||_2` 是残差向量的二范数。

## 8. 线性回归的目标

线性回归不是随便找一条直线，而是要找一组参数，让 MSE 尽可能小。

也就是：

```text
找到最合适的 w 和 b
让预测值 y_hat 尽量接近真实值 y
```

数学上可以写成：

```text
min ||y - Xw||_2^2
```

这就是最小二乘问题。

## 9. 与 MIT 线性代数第 4 章的关系

MIT 线性代数第 4 章的主线是：

```text
正交
-> 投影
-> 最小二乘
-> QR 分解
```

线性回归正好对应其中的“投影”和“最小二乘”。

从几何上看：

```text
Xw 是 X 的列向量的线性组合
所有可能的 Xw 构成 X 的列空间
真实标签 y 通常不在这个列空间里
线性回归要找列空间里离 y 最近的那个向量
```

这个最近的向量就是：

```text
y_hat = Xw_hat
```

残差：

```text
e = y - y_hat
```

会和 `X` 的列空间正交：

```text
X^T e = 0
```

代入：

```text
X^T(y - Xw_hat) = 0
```

得到正规方程：

```text
X^T X w_hat = X^T y
```

所以线性回归的本质不是“凭感觉画直线”，而是：

```text
在列空间中寻找 y 的正交投影。
```

## 11. 回归问题的评估指标

分类问题常用 accuracy。

回归问题不能用 accuracy，因为预测值是连续数值，通常不会和真实值完全相等。

常见回归指标：

```text
MAE   -> 平均绝对误差
MSE   -> 均方误差
RMSE  -> 均方根误差
R2    -> 决定系数，越接近 1 通常越好
```

当前阶段先记：

```text
MAE / RMSE 越小越好
R2 越接近 1 越好
```

## 12. 本节总结

```text
1. 线性回归用于预测连续数值。
2. 一元线性回归公式是 y_hat = wx + b。
3. 多元线性回归可以写成 y_hat = Xw。
4. 误差可以组成残差向量 y - Xw。
5. 线性回归要最小化残差向量的平方长度。
6. 从线性代数角度看，线性回归就是最小二乘和正交投影。
```
