# 线性回归第三课：正规方程

## 1. 这一节回答什么问题

前两节我们已经知道：

```text
线性回归要最小化损失函数
损失函数常用 ||y - Xw||_2^2
```

现在的问题变成：

```text
怎样直接求出让损失最小的参数 w？
```

这就是正规方程要解决的事。

## 2. 从目标函数开始

线性回归的目标是：

```text
min ||y - Xw||_2^2
```

这里：

```text
X   -> 特征矩阵
w   -> 参数向量
y   -> 真实标签向量
Xw  -> 预测向量
```

也就是说，我们想找一个 `w`，让预测向量 `Xw` 尽量接近真实向量 `y`。

## 3. 几何意义：投影到列空间

从线性代数角度看：

```text
X 的每一列 -> 一个特征列向量
Xw         -> 这些列向量的线性组合
```

所以所有可能的 `Xw` 都位于 `X` 的列空间 `Col(X)` 中。

真实标签 `y` 通常不在这个列空间中，因此 `Xw = y` 往往没有精确解。线性回归做的是：

```text
在 Col(X) 中找到离 y 最近的那个向量
```

记这个最优预测为：

```text
y_hat = Xw_hat
```

残差向量：

```text
r = y - y_hat
```

最优时，残差会与列空间正交：

```text
X^T r = 0
```

这就是正规方程的出发点。

## 4. 正规方程的推导结果

把：

```text
r = y - Xw_hat
```

代入正交条件：

```text
X^T(y - Xw_hat) = 0
```

展开：

```text
X^T y - X^T X w_hat = 0
```

整理得到：

```text
X^T X w_hat = X^T y
```

这就是正规方程。

如果 `X^T X` 可逆，那么可以写成：

```text
w_hat = (X^T X)^(-1) X^T y
```

这是一条直接求解最优参数的公式。

## 5. 为什么它叫“正规方程”

因为它不是原始方程：

```text
Xw = y
```

而是从最小二乘条件推导出来的新方程：

```text
X^T X w = X^T y
```

它是“最优解必须满足的方程”，所以叫正规方程。

## 6. 一元线性回归里的样子

假设只有一个特征，并把偏置项也写进矩阵：

```text
X_design =
[
  [1, x1],
  [1, x2],
  ...
  [1, xn]
]
```

参数向量：

```text
theta =
[
  b,
  w
]
```

那么预测就是：

```text
y_hat = X_design theta
```

正规方程变成：

```text
X_design^T X_design theta = X_design^T y
```

解出 `theta` 后，就同时得到了：

```text
b -> 截距
w -> 斜率
```

## 7. 正规方程和最小二乘的关系

这两个概念不是平行的，而是包含关系：

```text
最小二乘 -> 问题目标
正规方程 -> 解决最小二乘问题的一种推导结果
```

可以记成：

```text
我要让 ||y - Xw||_2^2 最小
-> 最优残差与列空间正交
-> 得到 X^T Xw = X^T y
```

## 8. 正规方程的优点和局限

优点：

```text
1. 不需要手动设置学习率
2. 对小规模特征问题很直观
3. 能直接体现最小二乘和线性代数结构
```

局限：

```text
1. 需要处理 X^T X
2. 特征很多时，矩阵求逆代价较高
3. X^T X 可能不可逆或数值不稳定
```

所以实际计算里，虽然公式常写成：

```text
w_hat = (X^T X)^(-1) X^T y
```

但数值计算通常更推荐：

```text
np.linalg.lstsq
QR 分解
SVD
```

而不是显式求逆。

## 9. 和 sklearn 的关系

在 `sklearn.linear_model.LinearRegression` 中，底层思想仍然是最小二乘，但通常不会让我们自己手动写逆矩阵。

也就是说：

```text
sklearn 帮你做了最小二乘求解
我们现在学的是它背后的数学结构
```

## 10. 本节总结

```text
1. 线性回归要最小化 ||y - Xw||_2^2。
2. 最优预测向量是 y 在 Col(X) 上的正交投影。
3. 最优残差满足 X^T(y - Xw_hat) = 0。
4. 由此得到正规方程 X^T Xw_hat = X^T y。
5. 正规方程是求最小二乘解的一种方法。
6. 实际计算中通常不推荐显式求逆，而更推荐 lstsq、QR 或 SVD。
```
