## 1. 这一节回答什么问题

前两节我们已经知道：

```text
线性回归要最小化损失函数
损失函数常用 ||y - Xw||_2^2
```

现在的问题变成：

```text
怎样直接求出让损失最小的参数？
```

这就是正规方程要解决的事。

## 2. 从目标函数开始

线性回归的目标可以写成：

```text
min_w ||y - Xw||_2^2
```

其中：

- `X`：特征矩阵
- `w`：参数向量
- `y`：真实标签向量
- `Xw`：预测向量

意思是：我们要找一个 `w`，让预测向量 `Xw` 尽量接近真实向量 `y`。

## 3. 一元线性回归的损失函数

一元线性回归模型：

```text
y_hat_i = w x_i + b
```

第 `i` 个样本的残差：

```text
e_i = y_i - y_hat_i
    = y_i - (w x_i + b)
```

如果有 `m` 个样本，平方误差和写成：

```text
SSE(w, b) = Σ_{i=1}^m (y_i - w x_i - b)^2
```

为了推导最优参数，通常把目标函数记成：

```text
J(w, b) = Σ_{i=1}^m (y_i - w x_i - b)^2
```

有些教材也会写成：

```text
J(w, b) = (1 / 2m) Σ_{i=1}^m (y_i - w x_i - b)^2
```

这两种写法的最优解位置相同，因为前面的常数不会改变最小值点。

下面为了推导更干净，我们使用：

```text
J(w, b) = Σ_{i=1}^m (y_i - w x_i - b)^2
```

## 4. 对 w 和 b 分别求偏导

要让 `J(w, b)` 最小，就要找驻点，也就是令偏导数等于 0。

### 4.1 对 w 求偏导

```text
dJ/dw
= d/dw [ Σ_{i=1}^m (y_i - w x_i - b)^2 ]
= Σ_{i=1}^m 2 (y_i - w x_i - b) (-x_i)
= -2 Σ_{i=1}^m x_i (y_i - w x_i - b)
```

令它等于 0：

```Latex
-2 Σ_{i=1}^m x_i (y_i - w x_i - b) = 0
```

两边同时除以 `-2`：

```text
Σ_{i=1}^m x_i (y_i - w x_i - b) = 0
```

展开：

```text
Σ_{i=1}^m x_i y_i
- w Σ_{i=1}^m x_i^2
- b Σ_{i=1}^m x_i
= 0
```

整理得到：

```text
w Σ_{i=1}^m x_i^2 + b Σ_{i=1}^m x_i = Σ_{i=1}^m x_i y_i
```

### 4.2 对 b 求偏导

```text
dJ/db
= d/db [ Σ_{i=1}^m (y_i - w x_i - b)^2 ]
= Σ_{i=1}^m 2 (y_i - w x_i - b) (-1)
= -2 Σ_{i=1}^m (y_i - w x_i - b)
```

令它等于 0：

```text
-2 Σ_{i=1}^m (y_i - w x_i - b) = 0
```

两边同时除以 `-2`：

```text
Σ_{i=1}^m (y_i - w x_i - b) = 0
```

展开：

```text
Σ_{i=1}^m y_i
- w Σ_{i=1}^m x_i
- m b
= 0
```

整理得到：

```text
w Σ_{i=1}^m x_i + m b = Σ_{i=1}^m y_i
```

## 5. 一元线性回归的正规方程组

于是得到关于 `w` 和 `b` 的两个方程：

```text
w Σ_{i=1}^m x_i^2 + b Σ_{i=1}^m x_i = Σ_{i=1}^m x_i y_i
w Σ_{i=1}^m x_i   + m b             = Σ_{i=1}^m y_i
```

这就是一元线性回归在标量形式下的正规方程组。

如果把它写成矩阵形式，可以写成：

```text
[ Σx_i^2   Σx_i ] [ w ] = [ Σx_i y_i ]
[ Σx_i       m  ] [ b ]   [   Σy_i   ]
```

如果把参数顺序改成 `[b, w]^T`，也可以写成：

```text
[   m      Σx_i   ] [ b ] = [   Σy_i   ]
[ Σx_i   Σx_i^2 ] [ w ]   [ Σx_i y_i ]
```

这两种写法只是参数排列顺序不同，本质完全一样。

## 6. 从标量方程到设计矩阵

黑马课程接下来会把一元形式过渡到统一矩阵形式。做法是把偏置项并入参数向量。

定义设计矩阵：

```text
X_design =
[
  [1, x_1],
  [1, x_2],
  ...
  [1, x_m]
]
```

定义参数向量：

```text
theta =
[
  b,
  w
]
```

于是：

```text
y_hat = X_design theta
```

损失函数统一写成：

```text
J(theta) = ||y - X_design theta||_2^2
```

## 7. 几何意义：投影到列空间

从线性代数角度看：

```text
X_design 的每一列    -> 一个列向量
X_design theta       -> 这些列向量的线性组合
```

所以所有可能的 `X_design theta` 都位于 `X_design` 的列空间 `Col(X_design)` 中。

真实标签 `y` 通常不在这个列空间中，因此：

```text
X_design theta = y
```

往往没有精确解。

线性回归真正做的是：

```text
在 Col(X_design) 中找到离 y 最近的那个向量
```

记最优预测为：

```text
y_hat = X_design theta_hat
```

残差向量：

```text
r = y - y_hat
```

最优时，残差和列空间正交：

```text
X_design^T r = 0
```

### 7.1 正交补充：这里的“正交”到底是什么意思

在向量空间里，两个向量 `u` 和 `v` 正交，最常用的代数判定是：

```text
u^T v = 0
```

几何上可以把它理解成“夹角是 90 度”，但在线性代数计算里，通常直接看点积是否为 0。

如果把矩阵的列写成：

```text
A = [a_1, a_2, ..., a_n]
```

那么：

```text
a_i^T a_j = 0 (i != j)  -> 不同列彼此正交
a_i^T a_i = ||a_i||_2^2 -> 自己和自己的点积等于长度平方
```

如果这些列不仅彼此正交，而且每一列长度都等于 1，就叫标准正交（orthonormal），此时：

```text
A^T A = I
```

这也是后面 QR 分解里 `Q` 的核心性质。

在线性回归里，`X_design^T r = 0` 不是一句抽象口号，而是可以拆成：

```text
X_design = [x_1, x_2, ..., x_n]

X_design^T r =
[
  x_1^T r,
  x_2^T r,
  ...,
  x_n^T r
]
```

所以 `X_design^T r = 0` 等价于：

```text
x_1^T r = 0
x_2^T r = 0
...
x_n^T r = 0
```

也就是残差 `r` 与 `X_design` 的每一列都正交，因此也与这些列张成的整个列空间 `Col(X_design)` 正交。

于是最小二乘的几何图像可以记成：

```text
y = y_hat + r
其中 y_hat 在 Col(X_design) 中
其中 r 垂直于 Col(X_design)
```

这就是“投影”的本质：`y_hat` 是落在列空间里的那部分，`r` 是投影后剩下的垂直误差。

这就是正规方程的出发点。

## 8. 矩阵形式下的正规方程推导

把

```text
r = y - X_design theta_hat
```

代入正交条件：

```text
X_design^T (y - X_design theta_hat) = 0
```

展开：

```text
X_design^T y - X_design^T X_design theta_hat = 0
```

整理得到：

```text
X_design^T X_design theta_hat = X_design^T y
```

这就是正规方程。

如果 `X_design^T X_design` 可逆，那么可以写成：

```text
theta_hat = (X_design^T X_design)^(-1) X_design^T y
```

如果不特别区分偏置和权重，而把参数统一记成 `w`，也常写成：

```text
X^T X w_hat = X^T y
w_hat = (X^T X)^(-1) X^T y
```

## 9. 为什么它叫“正规方程”

因为它不是原始方程：

```text
Xw = y
```

而是从最小二乘条件推导出来的新方程：

```text
X^T X w = X^T y
```

它表示“最优解必须满足的方程”，所以叫正规方程。

## 10. 一元线性回归的显式解

如果继续对一元线性回归的方程组做消元，还可以得到 `w` 和 `b` 的显式公式。

先由：

```text
w Σx_i + m b = Σy_i
```

解出：

```text
b = (Σy_i - w Σx_i) / m
```

再代回：

```text
w Σx_i^2 + b Σx_i = Σx_i y_i
```

整理后得到：

```text
w = [ m Σ(x_i y_i) - (Σx_i)(Σy_i) ] / [ m Σ(x_i^2) - (Σx_i)^2 ]
```

再带回去得到：

```text
b = (Σy_i - w Σx_i) / m
```

也可以写成均值形式：

```text
w = Σ[(x_i - x_mean)(y_i - y_mean)] / Σ[(x_i - x_mean)^2]
b = y_mean - w x_mean
```

这几组公式本质上都是同一个最小二乘解，只是写法不同。

## 11. 正规方程和最小二乘的关系

这两个概念不是平行关系，而是包含关系：

```text
最小二乘 -> 问题目标
正规方程 -> 求最小二乘解的一种推导结果
```

可以记成：

```text
我要让 ||y - Xw||_2^2 最小
-> 最优残差与列空间正交
-> 得到 X^T Xw = X^T y
```

## 12. 正规方程的优点和局限

优点：

```text
1. 不需要手动设置学习率
2. 对小规模特征问题很直观
3. 能直接体现最小二乘和线性代数结构
```

局限：

```text
1. 需要处理 X^T X
2. 特征很多时，矩阵求逆代价较高
3. X^T X 可能不可逆或数值不稳定
```

所以实际数值计算里，虽然公式常写成：

```text
w_hat = (X^T X)^(-1) X^T y
```

但更推荐：

```text
np.linalg.lstsq
QR 分解
SVD
```

而不是显式求逆。

## 13. 和 sklearn 的关系

在 `sklearn.linear_model.LinearRegression` 中，底层思想仍然是最小二乘，只是通常不会让我们自己手动去写逆矩阵。

也就是说：

```text
sklearn 帮你完成了最小二乘求解
我们现在学的是它背后的数学结构
```

## 14. 本节总结

```text
1. 线性回归要最小化 ||y - Xw||_2^2。
2. 一元线性回归可先写出 J(w, b) = Σ(y_i - w x_i - b)^2。
3. 对 w 和 b 分别求偏导并令其为 0，可得到一元线性回归的正规方程组。
4. 把偏置并入设计矩阵后，可以统一写成 X^T X theta = X^T y。
5. 最优预测向量是 y 在列空间上的正交投影。
6. 正规方程是求最小二乘解的一种方法。
7. 实际计算中通常不推荐显式求逆，而更推荐 lstsq、QR 或 SVD。
```
