线性回归的核心恰好就是第 4.2 和第 4.3 节：当方程 `Xw = y` 通常无精确解时，在 `X` 的列空间中寻找最接近 `y` 的向量 `Xw_hat`。

所以现在不需要暂停机器学习、回头学完整本线性代数。最合适的方式是：

```text

```

## 3. 一元线性回归：从直线到矩阵

一元线性回归写成：

```text
y_hat = wx + b
```

对 `m` 个样本统一预测时，可以把常数 `1` 作为一列加入特征矩阵：

```text
X =
[
  [1, x1],
  [1, x2],
  ...,
  [1, xm]
]

theta =
[
  b,
  w
]
```

于是所有预测可以统一写成：

```text
y_hat = X theta
```

形状关系是：

```text
X:       (m, 2)
theta:   (2, 1)
y_hat:   (m, 1)
```

这正是 NumPy 笔记里“行是样本、列是特征”和矩阵乘法的实际用途。

## 4. 多元线性回归：列的线性组合

有 `n` 个特征时：

```text
y_hat = w1*x1 + w2*x2 + ... + wn*xn + b
```

加入常数列以后仍然写成：

```text
y_hat = X theta
```

从线性代数角度看，`X theta` 是 `X` 的各列按照参数 `theta` 做线性组合。因此所有可能的预测向量都位于 `X` 的列空间 `Col(X)` 中。

```text
X 的一列 = 一个特征在所有样本上的取值
theta 的一个分量 = 该特征对应的权重
X theta = 所有特征列的加权和
```

## 5. 最小二乘就是投影

真实标签向量 `y` 通常不在 `Col(X)` 中，因此 `X theta = y` 往往没有精确解。线性回归改为寻找：

```text
让 ||y - X theta||_2^2 最小的 theta_hat
```

其中：

```text
e = y - y_hat = y - X theta_hat
```

叫作残差向量。最优预测 `y_hat` 是 `y` 在 `Col(X)` 上的正交投影，最优残差与列空间正交：

```text
X^T e = 0
```

代入 `e = y - X theta_hat`：

```text
X^T(y - X theta_hat) = 0
X^T X theta_hat = X^T y
```

这就是正规方程。

当 `X` 的列线性无关时，可以形式化写成：

```text
theta_hat = (X^T X)^(-1) X^T y
```

但实际数值计算通常不会显式求逆，而会使用 QR、SVD 或专门的最小二乘求解器。MIT 第 4.4 节的 QR 分解因此不是额外话题，而是更稳定地求解线性回归的一条路线。

## 6. 损失函数和二范数

平方误差和为：

```text
SSE = sum((yi - y_hat_i)^2)
    = ||y - X theta||_2^2
```

均方误差为：

```text
MSE = (1/m) * ||y - X theta||_2^2
```

它和 KNN 中的欧氏距离使用同一个二范数，但对象不同：

```text
KNN：          ||x_a - x_b||_2      比较两个样本向量
线性回归损失： ||y - X theta||_2    衡量整组预测残差
```

## 7. KNN 与线性回归的数学对比

| 角度 | KNN | 线性回归 |
|---|---|---|
| 预测目标 | 分类或连续值 | 连续值 |
| 核心对象 | 样本之间的距离 | 特征列的线性组合 |
| 数学表达 | `||x - x_i||` | `y_hat = X theta` |
| 学习方式 | 保存样本，预测时找邻居 | 训练时求参数 `theta` |
| 几何图像 | 特征空间中的局部邻域 | 把 `y` 投影到 `Col(X)` |
| 模型性质 | 局部、非参数模型 | 全局、参数模型 |

KNN 回归与线性回归虽然都预测连续值，但不是同一种“回归”：

```text
KNN 回归：用附近样本标签的平均值做局部预测
线性回归：用一组全局权重建立 X 到 y 的线性关系
```

## 8. 标准化与线性代数

对每一列特征做标准化：

```text
z_j = (x_j - mean_j) / std_j
```

可以理解为先平移坐标原点，再用对角缩放矩阵改变各坐标轴单位。设均值向量为 `mu`、标准差组成的对角矩阵为 `D`，则可以写成：

```text
X_scaled = (X - 1*mu^T) D^(-1)
```

它在线性回归中的作用需要和 KNN 区分：

```text
普通最小二乘：不标准化通常也能得到同一组预测，但系数尺度不同
梯度下降：标准化通常让不同方向的更新更均衡，收敛更快
岭回归/Lasso：标准化很重要，否则惩罚会不公平地作用于不同量纲的系数
KNN：标准化直接改变距离，通常会改变邻居和预测结果
```

## 9. Iris 二维可视化中的投影

Iris 原始样本位于四维特征空间 `R^4`。当前散点图只选取花瓣长度和花瓣宽度，相当于做坐标选择：

```text
[sepal_length, sepal_width, petal_length, petal_width]
                         -> [petal_length, petal_width]
```

这是一种从四维到二维的线性映射，但它与线性回归中的“正交投影到 `Col(X)`”不是同一个投影：

```text
绘图：选择两个坐标，目的是看得见
回归：寻找列空间中的最近向量，目的是最小化残差
```
