# 决策树第一课：基本概念

## 1. 决策树是什么

决策树是一类通过不断提出问题、逐步划分数据来完成预测的机器学习模型。

```text
是否适合打球？
天气是否晴朗？
├─ 是：湿度是否高？
│  ├─ 是：不打球
│  └─ 否：打球
└─ 否：打球
```

树中的每一次提问称为一次**分裂**；每个判断条件所在的位置是**内部节点**；最终给出预测结果的位置是**叶节点**。

## 2. 决策树能解决什么问题

- **分类**：预测离散类别，例如“是否是垃圾邮件”“用户是否流失”。
- **回归**：预测连续数值，例如“房价是多少”“销量是多少”。

分类树的叶节点输出类别；回归树的叶节点通常输出该节点样本目标值的平均值。

## 3. 核心目标：让数据更纯

每次选择分裂条件时，决策树都希望分裂后的子节点中样本类别尽量一致。

```text
分裂前：正类、负类混在一起
分裂后：左侧大多为正类，右侧大多为负类
```

节点中样本越属于同一类别，称该节点的“纯度”越高。后续会使用熵、信息增益和基尼系数来衡量纯度及分裂效果。

## 4. 决策树的优点与局限

优点：

- 规则清晰，容易解释和可视化。
- 可处理数值型与类别型特征。
- 通常不需要对特征进行标准化。

局限：

- 树过深时容易记住训练数据中的噪声，产生过拟合。
- 单棵树对数据变化较敏感。

## 5. 学习路线

1. 熵与信息增益：理解如何选择最佳分裂特征。
2. 基尼系数与 CART：学习 sklearn 常用的决策树思想。
3. 预剪枝与后剪枝：控制树的复杂度并防止过拟合。
4. sklearn 实战：训练、评估和可视化分类树、回归树。

## 6. 本课小结

决策树通过一系列可解释的条件判断进行预测；每次分裂都在追求让子节点的数据更纯。下一课学习：熵与信息增益。
